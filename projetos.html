Projeto Vovô Conectado

Site educativo desenvolvido com HTML, CSS e Bootstrap, voltado para inclusão digital de idosos. O projeto Vovô Conectado apresenta tutoriais ilustrados e acessíveis sobre o uso de aplicativos populares como WhatsApp, Facebook, Instagram, bancos e configurações do celular. Com foco em clareza e usabilidade, o site utiliza uma interface amigável, linguagem simples e layout responsivo para facilitar o aprendizado passo a passo.

Site StreamisNet

Site institucional responsivo desenvolvido com HTML, CSS e Bootstrap para a empresa fictícia Streamis Net, provedora de internet. O projeto apresenta seções como serviços, planos, sobre, depoimentos e contato, com layout moderno e adaptado para diferentes tamanhos de tela. Projeto cadastro de vilões

Sistema web desenvolvido com Python (Flask) e SQLite para gerenciamento de vilões e suas equipes. A aplicação conta com autenticação de usuário e permite operações de cadastro, edição, listagem e exclusão (CRUD) de vilões e equipes, vinculados por chave estrangeira. O projeto destaca o uso de rotas protegidas, organização modular com blueprints e persistência de dados com banco relacional.

ConvertaJá - conversor de moedas

Aplicação web desenvolvida com HTML, CSS, JavaScript e integração com a API pública Frankfurter.app para conversão de moedas em tempo real. O projeto ConvertaJá permite converter valores em Real (BRL) para Dólar (USD), Euro (EUR) e Libra (GBP) de forma prática e precisa. Conta com layout responsivo, interface acessível e validação de dados, sendo ideal para uso educacional ou cotidiano. O design foi personalizado com CSS puro, sem uso de frameworks visuais como Bootstrap.

API gerenciamento de produtos

API REST desenvolvida com Python (Flask) e SQLite para gerenciamento de produtos. A aplicação permite operações de cadastro, listagem, atualização e exclusão (CRUD) por meio de endpoints organizados. As rotas seguem os métodos HTTP adequados (GET, POST, PUT, DELETE) e incluem tratamento de erros para garantir respostas claras em caso de operações inválidas.
